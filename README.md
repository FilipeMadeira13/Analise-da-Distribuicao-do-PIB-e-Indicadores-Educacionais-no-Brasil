# Análise da Distribuição do PIB e Indicadores Educacionais no Brasil

Este projeto explora visualmente dados econômicos e educacionais do Brasil utilizando Python. O objetivo é investigar como o Produto Interno Bruto (PIB) está distribuído entre estados e regiões e analisar possíveis relações com indicadores educacionais.

A análise foi desenvolvida utilizando o Google Colab, permitindo que o notebook seja executado diretamente no navegador sem necessidade de instalação local.

## Objetivos da análise

* Explorar a distribuição do PIB entre estados brasileiros
* Comparar a participação econômica das regiões
* Visualizar a composição do PIB por setores
* Investigar relações entre indicadores educacionais e desempenho econômico

## Tecnologias utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn

## Visualizações desenvolvidas

O projeto utiliza diferentes tipos de gráficos para explorar padrões nos dados:

* Gráfico de pizza
* Gráfico de rosca
* Treemap
* Gráfico de cascata
* Scatterplot
* Bubble chart
* Slope chart

Essas visualizações permitem analisar tanto a **composição dos dados** quanto o **relacionamento entre variáveis**.

## Principais insights

A análise exploratória dos dados revela alguns padrões relevantes:

* A região Sudeste concentra a maior parcela do PIB brasileiro.
* Estados como São Paulo possuem participação econômica significativamente maior que a média nacional.
* Regiões Norte e Nordeste apresentam participação menor no PIB nacional.
* Indicadores educacionais apresentam relação moderada com o desenvolvimento econômico regional.

## Como executar o projeto

Este projeto foi desenvolvido no Google Colab.

1. Abra o notebook no Google Colab.

2. Execute as células sequencialmente para reproduzir toda a análise.

1. Clone o repositório


2. Instale as dependências

```
pip install pandas matplotlib seaborn
```

3. Execute o notebook em um ambiente Jupyter.
